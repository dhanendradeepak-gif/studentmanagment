# 🎓 Student Management System (Flask)

A simple web-based Student Management System built using **Flask (Python)**.  
This project allows users to add, view, and delete student records using a clean web interface.

---

## 🚀 Features

- ➕ Add new students (ID, Name, Age)
- 📋 View all students in a table
- ❌ Delete student records
- 💾 Data stored in JSON file
- 🌐 Simple web interface using Flask
- ⚡ Lightweight and beginner-friendly

---

## 🛠️ Technologies Used

- Python 🐍
- Flask 🌐
- HTML 🧾
- JSON 💾
